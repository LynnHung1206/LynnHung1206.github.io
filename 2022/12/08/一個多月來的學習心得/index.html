д»Ҡе№ҙ25жӯІз”ҹж—Ҙ,ж”¶еҲ°дҫҶиҮӘзӨҫзҫӨзҡ„зҘқзҰҸ,зҸҫеҜҰзҡ„зҘқзҰҸ,иЈңзҝ’зҸӯж–°зҡ„еҘҪжңӢеҸӢеҖ‘зҡ„зҘқзҰҸ(д»–еҖ‘йӮ„иІ·иӣӢзі•е№«жҲ‘ж…¶з”ҹе—ҡе—ҡе—ҡ),зңҹзҡ„жҳҜйқһеёёзҡ„ж„ҹеӢ•гҖӮ

жҲ‘жғід»Ҡе№ҙе№ҙеҲқеҒҡзҡ„йҖҷеҖӢиҪүиҒ·зҡ„жұәе®ҡзңҹзҡ„жҳҜйқһеёёеҘҪзҡ„жұәе®ҡ,еҫҲж…¶е№ёиҮӘе·ұеҒҡдәҶжұәе®ҡ,еҫҲй–ӢеҝғиҮӘе·ұе–ңжӯЎйҖҷжўқи·ҜгҖӮдёҚж•ўиӘӘжҲ‘иҮӘе·ұеҫҲеҺІе®і,дҪҶзўәеҜҰзҗҶи§ЈеҠӣдёҚйҢҜXDиҖҢдё”й–ӢеҝғеҲ°еҚҠеӨңдёҚзқЎиҰәзҲ¬иө·дҫҶеҜ«жүЈзҡ„зЁӢеәҰ,иҖҒеҜҰиӘӘжҲ‘иў«жҲ‘иҮӘе·ұеҡҮеҲ°дәҶгҖӮжҲ‘дёҖзӣҙд»ҘзӮәжҲ‘иҮӘе·ұжҳҜдёүеҲҶйҗҳзҶұеәҰзҡ„дәә,дҪҶеҺҹдҫҶдёҚжҳҜдёүеҲҶйҗҳзҶұеәҰ,иҖҢжҳҜжңүжІ’жңүжғіиҰҒеҒҡйҖҷд»¶дәӢгҖӮ

еҫһдёҖй–Ӣе§Ӣеӯёзҝ’еҲ°зҸҫеңЁ,еӨ§зҙ„еӯёдәҶе№ҫзЁ®жҠҖиЎ“,еҢ…еҗ«HTML5, CSS, JavaScript, Java SE, MySQL, JDBC, jQuery,еҢ…еҗ«е°ҲжЎҲиҰҸз•«й–Ӣзҷјд»ҘеҸҠзҸҫеңЁеңЁеӯёзҡ„servletжҠҖиЎ“еӣһй ӯзңӢеҺҹдҫҶжҲ‘е·Із¶“еӯёдәҶйҖҷйәјеӨҡдәҶ!дҪҶжҲ‘жҳҺзҷҪеӯёдәҶе’ҢжңғдәҶжҳҜ2еӣһдәӢ,е°Өе…¶жҳҜJSдёҖй–Ӣе§Ӣеҝ«иў«е®ғжҗһзҳӢеҲ°зҸҫеңЁжјёжјёзҝ’ж…Ј(зңӢжӣёзҡ„жҲҗжһңе—ҡе—ҡ),жҲ‘еҘҪеғҸжңүж„ҸиӯҳеҲ°иҮӘе·ұеҸҜиғҪжңүж…ўж…ўи®ҠеҺІе®і(дҪҶйӮ„жҳҜеҫҲиҸң)гҖӮ

еҫҲе°‘зҡ„жҷӮеҖҷжңғиҰәеҫ—з„Ұж…®,жҮ·з–‘иҮӘе·ұжҳҜдёҚжҳҜдёҚеӨ иӘҚзңҹ,дҪҶе…¶еҜҰиҮӘе·ұжё…жҘҡзҹҘйҒ“жҲ‘зҡ„ж©Ўзҡ®зӯӢе·Із¶“жӢүеҲ°дёҖеҖӢеәҰдәҶ,еҶҚжӢүдёӢеҺ»зҡ„зөҗжһңдёҚжҳҜжҲ‘жғіиҰҒзҡ„,ж–јжҳҜйӮ„жҳҜз…§и‘—иҮӘе·ұзҡ„зҜҖеҘҸгҖӮзқЎеүҚзҝ»2й ҒJavaжӣё,жңүз©әзңӢJSжӣё,ж•ҙзҗҶдёҖдёӢд№ӢеүҚеӯёйҒҺзҡ„,еҝғж…Ӣзӣ®еүҚиӘҝж•ҙеҫ—йӮ„дёҚйҢҜгҖӮдҪҶеҗҢеӯёеҘҪеғҸдёҚжҳҜйҖҷжЁЈзҡ„

жҲ‘жҳҺзҷҪжҷӮй–“дёҚеӨҡжңғжңүеЈ“еҠӣ,дҪҶдёҖзӣҙжғіи‘—йҖҷд»¶дәӢдёҰдё”зөҰдәҲзө„е“ЎеЈ“еҠӣдёҰдёҚжҳҜеҘҪдәӢ,еҗҢзҸӯзҡ„еҘҪжңӢеҸӢдҫҝжҳҜйҖҷжЁЈгҖӮжҲ–иЁұжҳҜиә«зӮәзө„й•·зҡ„еЈ“еҠӣе’ҢеҮәж–јжұӮеҘҪеҝғеҲҮзҡ„еҝғи®“д»–еҰӮжӯӨ,еёҢжңӣд»–иғҪеҘҪеҘҪиӘҝж•ҙйҒҺдҫҶгҖӮйҖҷжҷӮе…¶еҜҰеҫҲж…¶е№ёд№ӢеүҚзҡ„е·ҘдҪңз•¶йҒҺе°Ҹдё»з®Ў,йӣ–з„¶жІ’жҖҺйәјз®Ўдәә,дҪҶи·ҹи‘—еӨ§дё»з®Ўеӯёзҝ’еҲ°дәҶйқһеёёеӨҡдәӢжғ…,и®“жҲ‘йҖҷеҖӢеүҜзө„й•·(йҡұеҪўзө„й•·XD)зҹҘйҒ“жҖҺйәјжЁЈеҒҡдәӢдёҚжңғи®“дәәдёҚиҲ’жңҚгҖӮ

зёҪиҖҢиЁҖд№Ӣ,иЁҖиҖҢзёҪд№Ӣ,жҳҜй–Ӣеҝғзҡ„дёҖеҖӢеҚҠжңҲ(еҝ«иҰҒ2еҖӢжңҲдәҶ!),й–ӢеҝғиҮӘе·ұеӯёеҫ—й–Ӣеҝғ,й–ӢеҝғиҮӘе·ұдәӨеҲ°еҫҲеӨҡеҘҪжңӢеҸӢ,й–ӢеҝғиҮӘе·ұд»Ҡе№ҙж”¶еҲ°еҫҲеӨҡзҰ®зү©(е®№жҲ‘еҗҗж§ҪдёҖзүҮ650зҡ„е·§е…ӢеҠӣжҲ‘иҰҒдҫҶзңӢзңӢжҳҜд»ҖйәјзҘһд»ҷзҫҺе‘і),жҲ‘е§ӢзөӮзӣёдҝЎ,дәәз”ҹжңғйҒҺеҫ—и¶ҠдҫҶи¶ҠеҘҪгҖӮ
